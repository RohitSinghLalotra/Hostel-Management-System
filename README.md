# Hostel Management System

## Project Description
The Hostel Management System is a web-based application designed to simplify and automate hostel operations. It provides an efficient, centralized platform for managing rooms, students, bookings, and notices, eliminating manual processes and replacing them with a fast, reliable digital system.

## Objective
The primary goal of this project is to modernize hostel administration by providing a digital solution that manages room allocation efficiently, maintains accurate student records, streamlines the booking and approval process, and enables clear communication through notices. The system reduces human error and significantly improves overall administrative efficiency.

## Key Features

### Admin Features
- Add and manage rooms with capacity and pricing details  
- View all registered students in the system  
- Approve or reject room booking requests  
- Automatically assign students to rooms upon approval  
- Create and manage notices for student communication  

### Student Features
- Register using name and email  
- View assigned room details and capacity  
- Check roommate information  
- Book rooms easily through the platform  
- Track booking status (Approved / Rejected)  
- View latest hostel notices  

## Technologies Used

- **Backend:** FastAPI – High-performance Python web framework  
- **Validation:** Pydantic – Data validation and schema management  
- **Database:** PostgreSQL – Relational database for storing data  
- **Frontend:** HTML, CSS, JavaScript – User interface development
